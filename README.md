# 3urek4

> Caribbean software studio. Building solutions for the whole human race.

The official website for **3urek4** — a Jamaican software portfolio company building production-grade web and mobile products. The brand name draws from "Eureka" and the closing line of Jamaica's National Pledge.

🌐 **Live site:** [3urek4.vercel.app](https://3urek4.vercel.app)

## Tech Stack

- **Framework:** React 18, Vite
- **Styling:** Custom CSS with liquid-glass aesthetic, dark charcoal theme
- **Hosting:** Vercel

## Design System

- **Logo:** Asterisk motif on dark charcoal background
- **Aesthetic:** Liquid glass / glassmorphism with subtle depth
- **Typography:** Clean, minimal, system-led

## Portfolio Products

The studio's product portfolio includes:

- **Aegis** — AI-powered campus safety platform (Intellibus Social Good Award 2026)
- **Stack** — Multi-tenant inventory management SaaS
- **FreshJA** — Agricultural e-commerce platform
- **Ripple** — Mobile wellness app
- **Medic** — Healthcare queue management
- **SPARK** — Internal product framework
